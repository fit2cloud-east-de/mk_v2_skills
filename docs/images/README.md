# 图片资源（docs/images）

说明文档用的示意图放这里（不参与脚本执行）。在 [`README.md`](../../README.md) 等文件中可这样引用：

```markdown
![套件架构](docs/images/architecture/overview.png)
```

| 子目录 | 用途 |
|--------|------|
| [`architecture/`](architecture/) | 架构图：[`overview.png`](architecture/overview.png)（导出图，普惠体）· [`overview.html`](architecture/overview.html)（源文件，未强制引用字体） |
| [`workflow/`](workflow/) | 智能体/知识库工作流示意、节点链路、拓扑示意 |
| [`knowledge/`](knowledge/) | 知识库选型、分段/入库、MinerU 等流程示意 |
| [`screenshots/`](screenshots/) | MaxKB / 客户端操作截图（请打码密钥与内网地址） |
| [`misc/`](misc/) | 其它配图 |

建议 PNG / SVG / WebP；文件名可用英文小写 + 连字符（如 `kb-type-decision.svg`）。  
请勿放入含真实 API Key、密码或未脱敏数据的截图。
